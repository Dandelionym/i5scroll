# i5scroll

## 基本使用

```html
<script src="./js/jquery.min.js"></script>
<script src="./js/i5scroll.min.js"></script>
<script>
// 选择需要运动的元素，直接调用 i5Scroll() 即可
$(".scroll").i5Scroll();
</script>
```

## 一个参数

```javascript
// 默认 normal，支持 slow、fast
$(".scroll").i5Scroll({
    speed: 'normal'
});
```

## 我的特点

支持隐藏元素滚动
